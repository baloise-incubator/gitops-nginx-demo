More about our GitOps process

From the idea of a new product until the deployment in production - a matter of seconds, only by applying git commands!

  • - Automatically create new namespaces securely
  • - Decorate PullRequests using a ChatOps based Bot
  • - Preview environments for every PullRequest
Have a look at the tools that we use

Secure

Only allow namespace creation by a technical user - but fully automated whenever you start a new project or a preview namespace is created.

Cloud Native

Every ChatOps command triggers a Tekton pipeline, images get published to harbor and every single step runs in its own container.

Scalable

It works for a single team or a whole organisation of many agile DevOps teams, every team can operate independently using only git.

Open Source

Everything available as Open Source, all the components, tools and concepts. Feel free to use, contribute or fill issues. We love your feedback!

Tekton

Kubernetes-style native pipeline resources for declarative CI/CD.
